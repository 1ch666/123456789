部署方式

1. `index.html` 推到 GitHub Pages。 2. `cloudflare-worker/` 用 `wrangler deploy` 部署。 3. 把 Worker 網址加到頁面 `?api=`，或在瀏覽器 localStorage 記住。
如果你只想先本地開發，可以跑 `node server/server.js`，前端加上 `?api=http://localhost:3000`。
